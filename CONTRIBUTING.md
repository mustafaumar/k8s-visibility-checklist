# Contributing to k8s-visibility-checklist

Thank you for considering a contribution. This project is built for beginners, by someone learning. Contributions from people at any experience level are welcome.

## How to contribute
- Found a mistake? Open an issue.
- Want to add a checklist item? Submit a pull request with the command, what to look for, and at least one red flag.
- Something confusing? Open an issue describing what was unclear — that feedback is as valuable as a fix.

## Guidelines
- Keep language beginner-friendly — no jargon without explanation
- Every checklist item must include a working kubectl command
- Follow the [CNCF Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md)
